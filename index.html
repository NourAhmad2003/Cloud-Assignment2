What is CI/CD?

CI/CD is a software development practice that automates the integration, testing, and delivery of code. Continuous Integration (CI) merges code changes frequently, while Continuous Delivery (CD) ensures the software is always in a deployable state, reducing errors and speeding up release cycles.

My Pipeline

After pushing code to the main branch, GitHub Actions automatically triggers a workflow. This workflow checks for required files, builds the environment, and finally deploys the updated website to GitHub Pages, ensuring a seamless automated flow.
